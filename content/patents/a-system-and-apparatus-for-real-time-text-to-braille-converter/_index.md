- **Patent Number**: 202421066714
- **Filing Date**: August 14, 2024
- **Status**: Filed

### Description

This invention provides a real-time text-to-Braille converter designed to support visually impaired students with accessible educational tools. Combining affordability, user-friendliness, and low-maintenance operation, the system helps students interact with written content independently.

### Key Features

- **Components**: Includes an input-output module, processing module, and power supply for robust operation.
- **Adjustable Braille Output Speed**: Allows users to control Braille display speed according to individual preference.
- **Voice Activation**: Enables hands-free operation through voice commands, enhancing accessibility.
- **AI-Enabled Image Captioning**: Captures and analyzes images to generate Braille captions, making visual content accessible.
- **Teacher Monitoring Tool**: Stores Braille-converted content in a centralized database, enabling teachers to monitor and track student progress. Supports the development of a digital Braille library accessible via cloud storage.

This patent represents a significant advancement in assistive technology, fostering independence and accessibility in education for visually impaired students.
